## v2.3.7 更新日志

### 性能优化
- **Vault 图片加载大幅提速**：新增文件缓存机制，全库扫描一次后建立文件名索引，后续图片查找直接从缓存匹配，避免重复递归遍历
- **图片懒加载**：渲染后为所有图片添加 `loading="lazy"` 和淡入效果，减少首次打开文档时的卡顿

### 功能精简
- 移除导出长图片、导出 HTML、导出 DOC 功能，专注阅读体验
- 移除 Apache POI 依赖，APK 体积更小
- 保留表格/图表保存为图片功能

### 体验改进
- 字符统计对话框顶部显示当前文件名
- Vault 文件夹路径显示优化：显示文件夹名称而非完整 URI
- 阅读统计文案调整：「触达书籍」→「触达文件」，「阅读场次」→「阅读次数」

### Bug 修复
- 修复设置 Vault 文件夹后图片无法加载的问题（原查找逻辑仅搜索 .md 文件）
- 修复 `![[图片.png]]` 和 `![[路径/图片.png]]` 两种 wikilink 格式均无法加载图片的问题
