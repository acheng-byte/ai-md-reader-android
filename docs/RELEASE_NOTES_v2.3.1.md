## v2.3.1 更新日志

### 新功能

- **阅读统计**：新增阅读数据追踪，包括陪伴天数、活跃天数、阅读场次、累计阅读时长、触达书籍数，菜单中点击「阅读统计」查看
- **导出 DOC**：支持将当前文档导出为 DOC 格式，保存至 `Download/MD阅读器/DOC`
- **设置入口**：工具栏新增设置按钮，快速进入设置页面

### 优化

- **图片点击优化**：屏幕中央区域点击图片不再触发预览，改为打开显示设置面板，避免误触
- **统一版本号管理**：版本号集中到 `version.properties`，Android 构建自动读取，iOS 通过同步脚本更新
- **CI 并发控制**：GitHub Actions 添加 concurrency 配置，同一分支快速推送时自动取消旧构建，避免资源浪费
