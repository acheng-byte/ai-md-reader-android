## v2.3.1 更新日志

### 路径与链接修复
- 修复含斜杠路径的 wikilink/图片在 Vault 中找不到的问题（`[[子目录/文件]]`、`![[子目录/图片.png]]`）
- URL 编码改为按路径段编码，保留 `/` 分隔符，兼容更多 Android WebView 版本
- `findFile` / `findAssetInVault` 增加 URL 解码匹配、大小写不敏感匹配等多种回退策略
- 缓存构建时同时存储 URL 解码后的文件名，兼容不同设备的文件名编码差异
- `findFileInDir` 路径导航支持 URL 解码和大小写不敏感匹配

### 外链资源支持
- 修复 `![[https://...]]` 外链图片/视频/音频无法显示的问题
- 外部 URL 不再被错误地包装为 vault 内部路径

### 内部链接修复
- `.md` 文件链接编码优化，路径中的 `/` 不再被编码为 `%2F`
- 带 anchor 的链接（如 `文件.md#标题`）编码更准确

### Callout 渲染修复
- 修复 callout 标题正则吞掉正文内容（`breaks: true` 模式下换行导致匹配失败）
- 修复嵌套 blockquote callout 处理顺序错误（改为逆序遍历，先处理最内层）

### 其他修复
- 修复 `%%注释%%` 正则无法处理注释内含 `%` 的情况（如 `%% 50% done %%`）
- 修复本地 vault 图片扩展名检查未去除 query 参数（如 `file.png?v=2`）
- TOC 目录链接使用实际 heading ID，支持中键点击和复制链接
