## v2.2.14 更新日志

### Bug 修复
- 修复 extractDocId 日志刷屏问题（扫描时每个文件都触发一条日志，导致 4000+ 条重复记录）
- 修复日志查看器在 5000+ 条时渲染卡顿/白屏（改用 RecyclerView 虚拟列表，只渲染可见条目）

### 功能增强
- 日志查看器新增垂直滚动条，支持快速拖动定位
- 日志容量从 5000 条扩展到 10000 条
- Logger 新增 `getAllEntries()` / `getSummaryEntries()` 方法，支持按条目访问

### 性能优化
- 库文件夹扫描从两遍遍历优化为单遍遍历，大库扫描速度提升约一倍
