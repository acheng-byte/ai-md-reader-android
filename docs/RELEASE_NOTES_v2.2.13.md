## v2.2.13 更新日志

### Bug 修复
- 修复运行日志弹窗点击「全部」按钮无反应的问题（AlertDialog 自动关闭导致）
- 修复滑动浏览时误触图片预览的问题（触摸移动阈值从 10px 提升到 20px）
- 修复部分设备状态栏遮挡顶部工具栏、字符统计无法点击的问题（添加 `fitsSystemWindows`）

### 功能增强
- 运行日志新增 20+ 处操作记录：打开文档、收藏/取消收藏、标注开关/颜色/模式、保存/另存为、主题切换、护眼模式、Frontmatter、引用块、隐藏标题、字体选择、设置重置、库文件夹设置/清空、搜索库、历史记录删除/清空、编辑/预览模式切换
- 库文件夹扫描优化：两阶段扫描，先文本文件（MD/TXT）后其他文件（图片），大库搜索响应更快
- 启动优化：先打开文档，后台静默预扫描库文件夹，不阻塞 UI

### 其他
- iOS 端同步修复滑动误触图片预览问题
