# MD阅读器 v1.3

这是 **谁是专家** 发起的 MD阅读器 v1.3。本次为交互与可读性优化版本，让常用操作更顺手、阅读控件在各类机型上更易用。

## 本次更新

- 收藏 / 取消收藏移至工具栏常显，使用矢量星形图标（实心表示已收藏）。
- 工具栏常显项调整为四项：目录、源码 / 预览切换、收藏切换、更多菜单；“打开”移入更多菜单。
- 显示设置改为点击屏幕中央区域唤出（电子书阅读器常见手势），不再占用菜单项。
- 优化标题折叠箭头：内联到标题文字前（不再悬挂到最左、避免曲面屏边缘裁切），并放大、改用主题色，更易识别与点击。

## 使用影响

- 常用操作更集中：目录、模式切换、收藏、更多菜单都在工具栏常显位置。
- “打开文件”进入更多菜单，减少工具栏拥挤，更符合阅读时的优先级。
- 显示设置通过点击屏幕中央唤出，更接近电子书阅读器习惯。
- 曲面屏或窄屏上，标题折叠箭头不再贴边悬挂，降低被裁切和误触概率。
- 收藏状态通过实心 / 空心星标直接表达，当前文档是否收藏更清楚。

## 安装

下载 APK 后，在 Android 手机上允许“安装未知来源应用”并安装即可。

## 说明

- 默认不渲染 Markdown 中内嵌的原始 HTML。
- 远程图片需要联网才能显示。
- 从微信等应用打开的文件权限可能是一次性的，失效后历史记录会标注“授权过期”；若此前已收藏，可从收藏夹继续打开。
