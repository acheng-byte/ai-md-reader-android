## v2.3.0 更新日志

### Vault 搜索重构
- 全库文件缓存机制：打开文档时扫描一次建立文件名索引，后续查找 O(1) 命中
- `findFile` 改为按文件名全库搜索，忽略路径前缀（Obsidian 行为）
- 支持 `[[子目录/文件名.md]]` 无论实际路径如何都能找到

### 链接与图片修复
- 修复 `.md` 文件链接点击打开失败（被解析为 `/assets/` 路径）
- 修复链接/图片路径中含空格导致解析失败的问题（空格编码为 `%20`）
- 修复 `![[子目录/图片.png]]` 子目录图片查找失败

### 功能精简
- 移除导出长图片、导出 HTML、导出 DOC 功能，专注阅读体验
- 移除 Apache POI OOXML 依赖（保留 HWPF 用于 .doc 阅读）

### 体验改进
- 字符统计对话框顶部显示当前文件名
- Vault 文件夹路径显示优化：显示文件夹名称而非完整 URI
- 阅读统计文案：「触达书籍」→「触达文件」，「阅读场次」→「阅读次数」
- 图片懒加载 + 淡入效果，减少首次打开卡顿
