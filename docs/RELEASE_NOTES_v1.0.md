# MD阅读器 v1.0（历史补充记录）

这是 **谁是专家** 发起的 MD阅读器最早可安装版本记录。该版本奠定了项目的基础方向：在 Android 手机上轻量、离线、清楚地阅读 Markdown 文件。

## 本次更新

- 首个可安装 APK 版本。
- 支持打开本地 `.md` / `.markdown` 文件。
- 支持源码 / 预览两种阅读模式。
- 支持从微信等应用通过“用其他应用打开”进入。
- 支持字号、行间距、段间距调节。
- 支持跟随系统、浅色、深色主题。
- 内置 Markdown 渲染资源，常规阅读离线可用。

## 使用影响

- 解决手机上收到 `.md` 文件后只能当纯文本硬看的问题。
- 让 AI 生成的说明、会议纪要、代码片段和知识库内容可以在手机上直接预览。
- 明确项目初期取舍：先做阅读器，不做重型 Markdown 编辑器。

## 说明

- 这是 GitHub Release 的历史补充记录。
- 项目首次公开入库时已经迭代到 v1.1，因此 v1.0 没有单独保留 GitHub 源码快照或 Release APK。
- 需要下载使用时，建议安装最新版本。
