## 正文

大家好，我是 **谁是专家**。今天想正式把一个小工具公开出来：**MD阅读器**。

它是一款轻量级 Android Markdown 阅读器，主要解决一个很小、但越来越高频的问题：在手机上，怎么顺手打开并阅读一个 `.md` 文件。

过去 Markdown 更多出现在程序员的电脑里：README、技术文档、项目说明、代码笔记。但这两年情况变化很快。AI 对话、代码助手、知识库、会议纪要、自动化工作流，都会把内容直接产出成 Markdown。电脑上看这些内容很方便，可是一到手机上，经常就变成了另一回事。

微信里收到一个 `.md` 文件，不知道用什么打开；文件管理器里找到一份 Markdown 文档，只能当普通文本硬看；代码块、表格、标题层级、目录结构，在小屏幕上都不够舒服。

这不是一个宏大的问题，但它非常日常。

所以我做了 **MD阅读器**。它的目标很朴素：当你在微信、文件管理器、QQ、邮件或其他应用里遇到 Markdown 文件时，可以直接选择“用 MD阅读器 打开”，然后把它清楚、安静、离线地读起来。

我没有把它做成一个大而全的编辑器。第一阶段更重要的事情，是把“打开、阅读、跳转、排版”这些基础体验做好。

目前它已经支持：

- 打开本地 `.md` / `.markdown` 文件；
- 从微信、文件管理器、QQ、邮件等应用通过“用其他应用打开”进入；
- 在源码模式和预览模式之间切换；
- 在预览模式下一键复制代码块；
- 调整字号、行间距、段间距，并自动记住显示设置；
- 点击屏幕中央区域唤出显示设置，减少右上角菜单负担；
- 跟随系统浅色 / 深色主题，也可以手动切换；
- 根据标题自动生成文档目录，点击后快速跳转；
- 在预览模式下点击标题，折叠或展开对应章节；
- 记录最近打开历史，并区分“授权过期”和“已删除”；
- 收藏 Markdown 文件到应用私有目录，原文件删除或授权过期后仍可继续打开；
- 内置 Markdown 渲染和代码高亮资源，常规阅读无需联网。

我自己最常用的场景，是阅读 AI 生成的项目方案、代码说明、会议纪要、知识库片段、长提示词，以及各种从聊天里转发来的 `.md` 文件。很多内容不一定需要在手机上编辑，但很需要在手机上快速打开、快速扫读、快速定位重点。

MD阅读器就是为这个场景准备的。

它也做了一些克制的取舍。比如，预览默认不渲染 Markdown 里内嵌的原始 HTML，避免不可信内容进入阅读链路；核心渲染资源打包在 APK 内，尽量保证常规阅读不依赖网络；暂时不做复杂云同步、重型插件系统，也不急着做完整编辑器能力。

我希望它先成为一个可靠的小入口。

如果你平时会在手机上查看 AI 生成文档、代码笔记、会议纪要、产品说明、开源项目 README，或者经常从微信里收到 Markdown 文件，可以试试看。

当前公开版本是 **v1.3**，发布日期为 **2026 年 6 月 4 日**。相比首发版本，v1.2 增加了代码块复制、文件类型校验、历史状态区分和收藏夹；v1.3 进一步把收藏移到工具栏常显，优化了工具栏布局、显示设置唤出方式和标题折叠箭头。

这个产品不复杂，也不打算变得臃肿。它只是想把一个越来越常见的阅读动作做顺手一点。
