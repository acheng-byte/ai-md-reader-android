# MD阅读器 v1.1

这是 MD阅读器的首个开源发布版本。项目面向手机上的高频 Markdown 阅读场景，尤其是 AI 生成文档、代码说明、知识库片段、会议纪要和聊天转发文件。

## 亮点

- 支持本地 `.md` / `.markdown` 文件阅读。
- 支持从微信、文件管理器、QQ、邮件等应用通过“用其他应用打开”进入。
- 支持源码 / 预览两种模式。
- 支持字号、行间距、段间距、明暗主题调节。
- 新增文档目录，适合阅读长 Markdown。
- 新增标题折叠 / 展开。
- 新增打开历史与失效文件友好提示。
- 内置 markdown-it 与 highlight.js，常规阅读离线可用。

## 安装

下载 APK 后，在 Android 手机上允许“安装未知来源应用”并安装即可。

## 说明

- 默认不渲染 Markdown 中内嵌的原始 HTML。
- 远程图片需要联网才能显示。
- 从微信等应用打开的文件权限可能是一次性的，历史记录中再次打开时可能显示为无法访问。
