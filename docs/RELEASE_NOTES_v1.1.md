# MD阅读器 v1.1

这是 **谁是专家** 发起的 MD阅读器首个 GitHub 开源发布版本。项目面向手机上的高频 Markdown 阅读场景，尤其是 AI 生成文档、代码说明、知识库片段、会议纪要和聊天转发文件。

## 本次更新

- 在 v1.0 基础上新增文档目录，自动根据标题生成大纲。
- 支持点击目录标题平滑跳转，适合阅读长 Markdown。
- 支持预览模式下点击标题折叠 / 展开章节内容。
- 新增打开历史，可回到最近阅读过的文档。
- 对历史中无法访问的文件做友好提示。
- 保留源码 / 预览切换、字号 / 行距 / 段距、明暗主题等基础阅读能力。
- 内置 markdown-it 与 highlight.js，常规 Markdown 渲染和代码高亮离线可用。

## 使用影响

- 长文阅读不再只能从头滚动，可以通过目录快速定位。
- AI 生成的项目方案、技术说明、会议纪要等层级化文档更容易在手机上扫读。
- 标题折叠让用户可以先看结构，再展开重点章节。
- 打开历史让常读文件更容易回到上次入口。

## 安装

下载 APK 后，在 Android 手机上允许“安装未知来源应用”并安装即可。

## 说明

- 默认不渲染 Markdown 中内嵌的原始 HTML。
- 远程图片需要联网才能显示。
- 从微信等应用打开的文件权限可能是一次性的，历史记录中再次打开时可能显示为无法访问。
