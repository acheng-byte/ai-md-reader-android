# MD阅读器 v1.4

这是 **谁是专家** 发起的 MD阅读器 v1.4。本次新增「一键转发」，把阅读中的 Markdown 更顺手地分享给微信等社交应用里的联系人。该版本面向 Android / 支持 APK 安装的鸿蒙设备；iOS 暂不包含本次功能。

## 本次更新

- 新增「转发」：阅读时一键将**完整 `.md` 文档**通过系统分享面板转发到微信、QQ 等应用（如发送给微信好友）。
- 转发的是文档文件本身（经 FileProvider 临时授权共享），接收方得到可再次打开的 `.md`，而非被截断的纯文本。
- 转发入口常显在工具栏；应用本身不申请额外权限。

## 使用影响

- 读完一篇 AI 生成的方案 / 笔记 / 纪要后，可直接转发给微信好友或群，省去“导出再发送”的步骤。
- 受系统能力限制：App 只能把文件交给微信，最终由微信自身的「发送给朋友」界面选择联系人，无法在本应用内直接指定某个好友。

## 下载说明

- 本次为 **Android / 支持 APK 安装的鸿蒙设备** 更新；iOS 版暂不包含该功能。
- 安装包通过 GitHub Releases 分发，下载后在手机上允许“安装未知来源应用”即可侧载。
