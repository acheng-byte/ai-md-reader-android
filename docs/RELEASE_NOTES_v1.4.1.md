# MD阅读器 v1.4.1

这是 **谁是专家** 发起的 MD阅读器 v1.4.1。本次是 Android / 支持 APK 安装的鸿蒙设备的体验修复版，重点解决长文连续阅读和收藏历史可信度。

## 本次更新

- 新增阅读位置记忆：再次打开同一文档时，自动回到上次阅读处。
- 阅读位置按滚动比例保存，调整字号、行距、段距后仍能大致回到原位置。
- 修复已收藏文档在“打开历史”中误报“授权过期”的问题。
- 已收藏文档会直接从收藏夹本地副本打开，不再弹“从收藏夹打开”的额外提示；取消收藏后才回到原来源读取。

## 使用影响

- 读 AI 生成的长方案、长纪要、代码说明时，中途退出后可以更自然地接着读。
- 从微信等来源打开的临时授权文件，只要已经收藏，就不会因为原授权失效而在历史里显示不可用。
- v1.4 的一键转发完整 `.md` 文件能力继续保留。

## 下载说明

- 本次为 **Android / 支持 APK 安装的鸿蒙设备** 更新；iOS 版暂不包含该功能。
- 安装包通过 GitHub Releases 分发，下载后在手机上允许“安装未知来源应用”即可侧载。
