# MD阅读器 v1.2

这是 **谁是专家** 发起的 MD阅读器 v1.2 历史补充记录。基于 v1.1，本次更新聚焦“代码取用、文件选择可靠性、历史可信度”，并新增收藏夹。

## 本次更新

- 预览模式代码块新增一键复制按钮，方便在手机上取用 AI 生成的代码片段。
- 打开文件时仅允许选择 `.md` / `.markdown`，选择其他类型会有友好提示。
- 打开历史区分两种失效状态，不再笼统标注：
  - 「授权过期」：如微信等一次性授权失效，文件可能仍在但暂时无法访问；
  - 「已删除」：文件已被物理删除。
- 新增收藏夹：
  - 收藏会把文档复制一份到应用收藏目录，原文件被删除或授权过期后仍可从收藏夹打开；
  - 以原始文件为身份去重，同一文件只保留一份副本，反复收藏 / 取消不会重复复制；
  - 取消收藏会同步删除该副本。

## 使用影响

- 手机阅读 AI 生成代码时，可以直接复制代码块，不必在小屏里手动框选。
- 文件选择更可靠，误选非 Markdown 文件时不会进入混乱阅读状态。
- 历史记录的失效原因更清楚，用户能判断是“文件没了”还是“微信等来源授权过期”。
- 收藏夹把重要 Markdown 复制到应用私有目录，降低外部文件失效对阅读的影响。

## 下载说明

- v1.2 与 v1.3 是本地连续迭代后同步到 GitHub 的版本。
- GitHub 上不单独保留 v1.2 的安装包；该 Release 用于补齐版本更新记录。
- 需要安装使用时，建议下载 v1.3 或更新版本。

## 说明

- 默认不渲染 Markdown 中内嵌的原始 HTML。
- 远程图片需要联网才能显示。
- 收藏副本以 UTF-8 文本形式保存在应用私有目录，卸载应用会一并清除。
- v1.2 的 GitHub 标签只用于版本记录补充，不代表单独保存了一份 v1.2 源码快照。
