# MD阅读器 v2.2.9

## 修复

- **修复 Vault 文件夹含中文时搜索失败的根因**：URI 中的中文字符未被 percent-encode，导致 Content Provider 匹配失败，所有文件查找返回空。现在自动规范化编码。
- **修复递归列目录在子目录断裂的问题**：DocumentsContract 回退创建的子 DocumentFile 缺少 tree URI 上下文，递归到第二层时全部失效。现在所有递归方法传递原始 treeUri，确保任意深度都能正常工作。

## 改进

- 日志缓冲区从 500 条扩大到 5000 条
- 日志消息改为中文说明，用【标签】格式标注操作类型，便于诊断
